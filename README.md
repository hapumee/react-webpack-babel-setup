## React Webpack Babel Project Setup

### Features
- React 16  
- Webpack 4  
- Babel 7  
- Hot Module Replacement

### Installation
- <code>git clone git@github.com:hapumee/react-webpack-babel-setup.git</code>
- cd react-webpack-babel-setup
- npm install
- npm start
- on browser: <code>http://localhost:8080</code>
